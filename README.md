# Lite-Login

这是一个Vue3+Ts的前端模板页面，采用Pinia状态管理组件，Element-Plus界面组件

Element-Plus组件自动按需导入，无需做额外配置。

目前已实现了登录，注册，修改密码三个界面。

默认使用Axios作为Http组件，但未实现相应细节，留给使用者自行开发。

适合快速启动一个前端模板小项目，主要是为了应付学习的各种作业。

### 安装依赖

```powershell
yarn install
```

### 运行项目

```powershell
yarn serve
```

### 构建项目
```powershell
yarn build
```
